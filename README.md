# LightScan

ユーザーランドで動作する簡易的なRustで書かれたメモリスキャナー?です。WindowsAPIを呼んでるだけ。
適切に保護されている最近のゲームにはどう考えても効かないので、自己責任で使うといいと思います。実行には管理者権限が必要です。

## UnrealEngine

リフレクションをもとにダンプしたメモリからクラス、プロパティ、インスタンスなどを取得できます。
シェルコードをターゲットプロセスに書き込み、ProcessEventを発火させています。
Blueprint関数も呼ぶことが可能です。

オフセットなどがUE5.5での決め打ちなので、バージョンによっては動かないと思います。
